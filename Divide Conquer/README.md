# Divide & Conquer (23.04.2020)
## 분할정복이란?
1. 분할 : 해결할 문제를 여러개의 작은 부분으로 나눔
2. 정복 : 나눈 작은 문제를 각각 해결
3. 통합(필요하다면) : 해결된 해답을 모음
(degenerate case 고려)

### Basic Theorem
1. 병합정렬 Merge Sort
2. 퀵정렬 Quick Sort
